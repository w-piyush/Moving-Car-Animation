# # 🚗 Moving Car Animation

A smooth and visually appealing **Moving Car Animation** built using HTML, CSS, and JavaScript. This project demonstrates animation techniques, motion effects, and responsive design.


## 🚀 Live Demo
 - Unmute and play the video for sound effects.


https://github.com/user-attachments/assets/57cea508-395e-4937-9fcf-cf633af9c084



## ✨ Features

* 🚗 Continuous car movement animation
* 🌄 Moving background for realistic effect
* 🔊 Engine sound effects
* ⚡ Smooth transitions using CSS animations
* 💻 Clean and minimal UI




## 🛠 Tech Stack

* HTML
* CSS
* JavaScript



## ⚙️ How It Works

* CSS keyframes are used to animate the car and background
* Layers (car, road, background) move at different speeds to create a **parallax effect**
